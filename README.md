# Employee-Performance-Analysis-Prediction
#  Employee Performance Analysis & Prediction

##  Overview
Employee performance is a critical factor in organizational success. This project analyzes employee data from **INX Future Inc.** to uncover performance patterns, identify key influencing factors, and build a machine learning model to predict employee performance.

The solution enables **data-driven HR decisions**, helping organizations improve productivity, employee satisfaction, and hiring quality.


##  Objectives
-  Analyze **department-wise performance trends**
-  Identify **top factors impacting employee performance**
-  Build a **predictive machine learning model**
-  Provide **actionable business recommendations**


##  Dataset
The dataset contains structured employee information, including:

-  Demographics: Age, Experience
-  Organizational: Department, Job Role
-  Satisfaction Metrics: Environment Satisfaction, Work-Life Balance
-  Target Variable: **Performance Rating**

##  Project Workflow

###  Data Preprocessing
- Handled missing values
- Encoded categorical variables
- Cleaned and structured dataset

###  Exploratory Data Analysis (EDA)
- Department-wise performance comparison
- Correlation heatmap
- Feature distribution analysis

###  Feature Engineering
- Selected relevant features
- Removed redundant data

###  Model Building
- Algorithm: **Random Forest Classifier**
- Train-Test Split: 80/20
- Model trained on processed dataset

###  Model Evaluation
- Accuracy Score
- Feature Importance Analysis


##  Key Insights

###  Department Performance
- Performance varies significantly across departments
- Some departments consistently underperform

###  Top 3 Factors Affecting Performance
-  Environment Satisfaction  
-  Work-Life Balance  
-  Experience Level  

###  Observations
- Employees with higher satisfaction levels show better performance
- Poor work-life balance negatively impacts productivity
